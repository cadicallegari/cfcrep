Máquina Virtual MIPS Monociclo 

Autores:
	Claudinei F. Callegari
	Lucas M. Comin

	Trabalho desenvolvido para a disciplina Arquitetura e Organização de Computadores
da UNIOESTE (Universidade Estadual do Oeste do Paraná).

	A máquina foi desenvolvida na linguagem Java com jdk6 no sistema operacional
GNU/Linux, com interface gráfica em Swing.

	Seu funcionamento consiste em carregar um arquivo contendo instruções em binário
e ao pressionar o botão executar é simulado então um ciclo de clock da arquitetura MIPS monociclo.

	Para executar o programa é necessario ter instalado JVM  >= 6. Execute o 
comando no diretorio que possui o arquivo maquina_virtual.jar
		
		$ java -jar maquina_virtual.jar

	No menu Exibir é possivel ver detalhes da memória e dos registradores da máquina.
